1. Project Title

Auto Sales — Exploratory Data Analysis

2. Short Description / Purpose

Exploratory Data Analysis on an Auto Sales dataset to uncover revenue patterns, customer behaviour, and market trends — with feature engineering prepared for future ML modeling.

3. Tech Stack

Python · Pandas · NumPy · Matplotlib · Seaborn

4. Features & Highlights
- Data Cleaning — datetime conversion, whitespace stripping, duplicate & null checks
- Univariate & Bivariate Analysis — sales distribution, product line performance, deal sizes
- Time Series Analysis — monthly sales trends and day-of-week patterns
- Geographic Insights — top 10 countries by revenue
- Feature Engineering — sales buckets, recency buckets, customer value segments, discount features
- Correlation Heatmap — relationships across key numerical features
